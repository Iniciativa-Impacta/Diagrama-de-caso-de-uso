Lib Virtual - Plataforma de Assinatura de E-books

💡 Como exportar para o Miro:

  1. Clique com botão direito em cada diagrama
  2. Selecione "Salvar imagem como..." para exportar como SVG/PNG
  3. No Miro, use "Importar imagem" ou "Pegar imagem" para adicionar os diagramas ao quadro
  4. Alternativamente, copie os códigos Mermaid e use a integração do Miro com Mermaid

📱 Para adicionar ao Miro:

  1. Método 1 (Recomendado): Abra o Miro → New board → Apps → Procure por "Mermaid" → Cole os códigos dos diagramas
  2. Método 2: Use Print Screen para capturar os diagramas e copiar como imagem para o Miro
  3. Método 3: Clique direito em cada diagrama → "Copiar imagem" → Cole no Miro como um quadro colaborativo
  4. Método 4: Exporte este HTML como PDF e importe no Miro como referência
